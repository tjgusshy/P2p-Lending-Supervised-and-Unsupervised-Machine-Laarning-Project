P2P Lending Risk Analysis & Investor DashboardProject OverviewThis repository contains an end-to-end machine learning pipeline designed to analyze Peer-to-Peer (P2P) lending data. The project aims to classify borrower risk, calculate the Probability of Default (PD), and cluster borrowers to help investors optimize their portfolios. The final output is visualized in an interactive Investor Dashboard
